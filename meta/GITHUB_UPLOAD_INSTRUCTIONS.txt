================================================================================
TROUBLESHOOTING
================================================================================

Problem: "git is not recognized as an internal or external command"
Solution: Install Git from https://git-scm.com/download/win and restart computer

Problem: "fatal: remote origin already exists"
Solution: Run this command first:
  git remote remove origin
Then try again from Step 10

Problem: "fatal: not a git repository"
Solution: Make sure you're in the correct folder:
  cd "C:\Users\VENKATESH\OneDrive\Desktop\meta"
Then run from Step 3

Problem: "fatal: 'origin' does not appear to be a 'git' repository"
Solution: Make sure you ran Step 10 Command 1 correctly. Check you replaced YOUR_USERNAME

Problem: "Permission denied (publickey)" when pushing
Solution: You need to set up SSH key or use HTTPS token. GitHub will show you options.

Problem: "Repository not found"
Solution: Make sure you:
  1. Created the repository on GitHub first (Step 8)
  2. Used the correct GitHub username (not email)
  3. Repository is PUBLIC
